# 📥 DualWrite 评论录入助手 · 安装说明

## 安装步骤

1. **下载插件文件夹**，解压到本地某个位置（不要删除这个文件夹）

2. **打开 Chrome 扩展管理页**
   - 地址栏输入：`chrome://extensions/`
   - 或者：右上角三点菜单 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 右上角找到「开发者模式」开关，打开它

4. **加载插件**
   - 点击「加载已解压的扩展程序」
   - 选择这个文件夹（里面有 manifest.json 的那层）

5. **完成！** 插件图标会出现在浏览器右上角

---

## 使用方法

### 录入评论
1. 打开小红书任意页面
2. 用鼠标**划选**评论文字（超过3个字才会触发）
3. 出现红色「📥 录入」按钮，点击录入
4. 点「📁 选分类」可以切换目标分类

### 导入到工作站
1. 点击浏览器右上角插件图标
2. 查看待导入列表
3. 点「📤 生成导入代码」
4. 点「📋 一键复制代码」
5. 打开工作站网页
6. 按 **F12** 打开开发者工具 → 点「Console（控制台）」标签
7. 粘贴代码，按回车
8. 弹出「导入成功」提示后，**刷新工作站页面**即可看到新评论

---

## 注意事项
- 插件文件夹不能删除，删了插件就失效了
- 如果小红书更新导致插件异常，检查 content.js 是否报错
- 图标文件（icons/）缺失不影响功能，只是没有图标显示

## 文件说明
```
manifest.json   插件配置
content.js      注入小红书页面的脚本
content.css     注入的样式
popup.html      点插件图标弹出的面板
popup.js        面板逻辑
background.js   后台服务
```
